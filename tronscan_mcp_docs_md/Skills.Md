

# TronScan Skills

**TronScan Skills** is a collection of Agent skills built on the [TronScan MCP Server](/en/mcp), enabling AI Agents (such as Claude) to query real-time TRON blockchain data using natural language — no manual API calls required.

---

## Skills List

| Skill | Directory | Description |
| --- | --- | --- |
| **Token Scanner** | [tronscan-token-scanner](/en/skills/tronscan-token-scanner) | TRX/token price, supply, market cap, rating, holdings |
| **Account Profiler** | [tronscan-account-profiler](/en/skills/tronscan-account-profiler) | Address assets, tokens, DeFi, energy, votes |
| **Contract Analysis** | [tronscan-contract-analysis](/en/skills/tronscan-contract-analysis) | Contract verification, popular methods, top callers |
| **Transaction Info** | [tronscan-transaction-info](/en/skills/tronscan-transaction-info) | Transaction result, confirmation status, resource usage |
| **Block Info** | [tronscan-block-info](/en/skills/tronscan-block-info) | Latest block, block producer, TRX burned |
| **Token List** | [tronscan-token-list](/en/skills/tronscan-token-list) | TRC20 list, trending tokens, 24h volume |
| **Data Insights** | [tronscan-data-insights](/en/skills/tronscan-data-insights) | Daily active accounts, top accounts, trending tokens/contracts |
| **Realtime Network** | [tronscan-realtime-network](/en/skills/tronscan-realtime-network) | TPS, node count, TVL, total transactions |
| **Search** | [tronscan-search](/en/skills/tronscan-search) | Resolve token/contract address by name |
| **SR Governance** | [tronscan-sr-governance](/en/skills/tronscan-sr-governance) | Super Representatives (SR), candidates, votes, chain parameters & proposals |
| **Stablecoin Analysis** | [tronscan-stablecoin](/en/skills/tronscan-stablecoin) | Supports USDT, USDC, USDD, and TUSD |
